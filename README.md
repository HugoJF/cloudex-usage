# Claudex Usage

Claudex Usage is a GNOME Shell panel surface for seeing Claude Code and Codex usage
windows and reset times without leaving the current workflow.

The project has a developer-installable static primitive catalog for its approved
Direction D design. No provider integration or production extension exists yet.
